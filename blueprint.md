# Project Blueprint: Lotto Number Generator

## Overview

This project is a simple, visually appealing web application that generates random lottery numbers. It's built with modern HTML, CSS, and JavaScript, focusing on a clean user experience and a modern aesthetic.

## Design and Features

### Style and Design

*   **Layout:** A centered, card-based UI that is responsive and works well on mobile and web.
*   **Color Palette:** A gradient background of blues and purples, with a white container for the main content. The lottery balls are color-coded by number range.
*   **Typography:** Clean and modern sans-serif fonts.
*   **Effects:**
    *   A subtle background texture to add a premium feel.
    *   Multi-layered drop shadows on the main container to create depth.
    *   A "glow" effect on the "Generate" button when hovered.
*   **Iconography:** A slot machine emoji in the title to enhance the theme.

### Features

*   **Random Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Visual Feedback:** The generated numbers are displayed in colored "balls" for easy reading.
*   **Color-Coded Numbers:**
    *   1-10: Yellow
    *   11-20: Blue
    *   21-30: Red
    *   31-40: Gray
    *   41-45: Green
*   **One-Click Generation:** A simple button to generate a new set of numbers.
*   **Theme Toggle:** A switch to toggle between light and dark modes. The user's preference is saved in local storage.
*   **Affiliate Inquiry Form:** A simple Formspree-powered contact form for affiliate inquiries, collecting name, email, and message.
